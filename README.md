# japanese_vocab_tester
# 日语词汇测试程序 🎌

一个简单的网页版日语词汇测试工具，帮助初学者记忆基础日语词汇。

## 功能特点

-   随机显示中文意思，测试日语词汇掌握程度
-   即时反馈正确与否
-   简洁易用的界面

## 如何使用

1.  直接访问 [GitHub Pages页面](https://你的用户名.github.io/仓库名/)
2.  查看屏幕上显示的中文意思
3.  在输入框中输入对应的日语词汇
4.  点击"提交"按钮查看结果

## 本地运行

克隆本项目后，直接双击打开`index.html`文件即可在浏览器中运行。
